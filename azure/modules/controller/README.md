# TrinityX Terraform Controller Module

- [ ] This module will perform below operation for TrinityX project:
- [ ] Create TLS Public and Private key for SSH Authentication for the Controller.
- [ ] Store SSH keys to the Azure SSH keys.
- [ ] Create a Public IP Address for the Controller, to access it from the outside.
- [ ] Create Network Interface for the Controller.
- [ ] Create Controller Virtual Machine.


- [ ] For More details for variables, visit the variable file [variables.tf](variables.tf)
